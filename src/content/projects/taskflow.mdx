## 프로젝트 개요

TaskFlow는 교내 해커톤에서 시작된 팀 협업 도구입니다. 팀 프로젝트를 하면서 느꼈던 "작업 현황 공유의 어려움"을 해결하기 위해 만들었습니다.

## 주요 기능

### 실시간 칸반 보드
- 드래그 앤 드롭으로 작업 상태 변경
- Supabase Realtime으로 팀원 간 실시간 동기화
- 작업 우선순위 및 담당자 지정

### 팀 채팅
- 프로젝트별 채팅방
- 파일 공유 지원
- 멘션(@) 기능

### 마감일 관리
- 캘린더 뷰
- 마감 임박 알림
- 주간/월간 통계

## 기술적 도전

### Supabase Realtime 활용
칸반 보드의 실시간 동기화를 위해 Supabase Realtime을 사용했습니다. 여러 사용자가 동시에 카드를 옮길 때 발생하는 충돌을 `updated_at` 타임스탬프를 기반으로 해결했습니다.

### dnd-kit을 선택한 이유
`react-beautiful-dnd`가 더 유명하지만, 번들 사이즈와 접근성(a11y) 지원 면에서 `dnd-kit`이 더 나았습니다.

## 배운 점

- 실시간 데이터 동기화에서의 충돌 해결 전략
- Supabase RLS(Row Level Security)를 활용한 권한 관리
- 팀원 피드백 기반 이터레이션의 중요성
