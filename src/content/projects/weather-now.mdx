## 프로젝트 개요

WeatherNow는 OpenWeather API를 활용한 날씨 대시보드입니다. 현재 위치의 날씨 정보와 주간 예보를 Chart.js 그래프로 시각화합니다.

## 주요 기능

- 현재 날씨 정보 (온도, 습도, 풍속, 체감온도)
- 주간 기온 변화 그래프
- 도시 검색 및 즐겨찾기
- 다크 모드 지원
- PWA 지원 (모바일 홈 화면 추가)

## 기술적 포인트

### API 호출 최적화
무료 API 한도(분당 60회) 내에서 운영하기 위해 요청 결과를 로컬 스토리지에 캐싱하고, 10분 단위로 갱신하도록 설계했습니다.

### Chart.js 커스터마이징
기본 차트 스타일을 커스터마이징하여 다크 모드에서도 가독성 좋은 그래프를 구현했습니다.
