## 프로젝트 개요

DevLog는 개발자를 위한 기술 블로그 플랫폼입니다. Markdown에 React 컴포넌트를 삽입할 수 있는 MDX를 활용하여, 인터랙티브한 기술 글을 작성할 수 있습니다.

## 주요 기능

### MDX 지원
- Markdown 문법으로 글 작성
- React 컴포넌트 삽입 가능
- frontmatter로 메타데이터 관리

### 코드 하이라이팅
- Shiki 기반 구문 하이라이팅
- 30개+ 언어 지원
- 라인 넘버 표시

### SEO 최적화
- 자동 메타태그 생성
- sitemap.xml 자동 생성
- OpenGraph 이미지 지원

## 기술적 도전

Next.js의 정적 사이트 생성(SSG)을 활용하여 빌드 타임에 모든 글을 사전 렌더링합니다. 이를 통해 Lighthouse 성능 점수 95+를 달성했습니다.

## 배운 점

- 정적 사이트 생성의 장단점
- SEO 최적화 전략
- MDX 파서 커스터마이징
