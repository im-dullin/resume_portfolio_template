## 프로젝트 개요

FoodMap은 공공데이터 포털의 음식점 정보와 카카오맵 API를 결합한 맛집 추천 서비스입니다. 공공데이터 활용 경진대회 출품작으로, 사용자의 현재 위치를 기반으로 주변 맛집을 추천합니다.

## 주요 기능

### 위치 기반 추천
- Geolocation API로 현재 위치 감지
- 카카오맵에 맛집 마커 표시
- 거리순/평점순 정렬

### 사용자 리뷰
- 별점 + 텍스트 리뷰 작성
- 사진 업로드 (Firebase Storage)
- 리뷰 좋아요 기능

### 북마크
- 나만의 맛집 리스트 저장
- 카테고리별 분류
- 공유 기능

## 기술 스택 선택 이유

### React + Styled Components
컴포넌트 단위로 스타일을 관리하기 위해 Styled Components를 선택했습니다. 테마 시스템을 활용해 다크 모드도 지원합니다.

### Firebase
인증, 데이터베이스, 스토리지를 한 곳에서 관리할 수 있어 빠르게 MVP를 만들 수 있었습니다.

## 성과
- 공공데이터 활용 경진대회 장려상 수상
- 200개+ 음식점 데이터 수집 및 정규화
